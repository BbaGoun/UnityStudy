**이동 키 :** wasd
* 이동 키를 빠르게 두 번 누르면 달립니다.

**점프 키 :** Space

**상호작용 키 :** e
* 상호작용 가능한 오브젝트는 주황색 테두리가 생깁니다.

**공격 조준 :** 마우스

**공격 키 :** 마우스 왼쪽 클릭
* 망치(무기) 가 있어야 공격이 가능합니다.

**구르기 :** 마우스 오른쪽 클릭
* 구르는 동안은 무적입니다.

**인벤토리 열기 :** i 또는 우측 하단의 가방 아이콘 클릭

**Option 열기 :** ESC 또는 우측 하단의 톱니 아이콘 클릭

맵에 2개의 상자가 있습니다.
아이템을 얻은 후, 3마리의 적을 쓰러트리세요.
적을 모두 쓰러트린 후, 트로피와 상호작용하여 게임을 승리하세요.
